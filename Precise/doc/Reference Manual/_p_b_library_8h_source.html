67 /* Used to enable/set/get the current library log level.
68  * Logging is always switched off by default but can be turned on at runtime
69  * to get diagnostic information also after the app has been deployed.
70  * Logging messages are sent to the XCode console output, Apple System Log and
71  * stored as a log file in the consuming app sandbox at:
72  * ./Library/Caches/Logs/PreciseBiometricsiOSLibrary.log. The max size of the
73  * log file is 1 MB.
74  * An additional new file is created with filename
75  * PreciseBiometricsiOSLibrary <n>.log for each fresh start of the app or if
76  * the log file size exceeds 1 MB.
77  * Note that logging adds some minimal processing and memory overhead.
78  * The 'Verbose' log setting include information that may be sensitive to the
79  * application such as APDU traces. This setting should not be used unless
80  * absolutely required (or during development/troubleshooting).
81  */
