40 /* The PBAccessory class delivers specific notifications for when the Tactivo gets connected
41  * and is ready for communication and when it gets disconnected. For more detailed information
42  * about accessory connections, use EAAccessoryDidConnectNotification (EAC) and
43  * EAAccessoryDidDisconnectNotification (EAD) notifications. Note that an EAC notification is
44  * received directly after the device is inserted into the Tactivo, but that the associated
45  * accessory does not contain the Tactivo protocol strings. When the Tactivo is authenticated
46  * after 1-2 seconds an EAD notification is sent, immediately followed by a new EAC. This new
47  * EAC will then have an associated accessory containing the correct Tactivo protocol strings,
48  * meaning that the Tactivo is now ready for communication. Also note that if the Tactivo is
49  * removed from the device before the authentication is completed, no EAC with an associated
50  * accessory with the Tactivo protocol strings will be sent. */
