Detailed Description

Class that handles communication with the Tactivo accessory that is not biometry or smartcard related.

PBAccessory generates the following notifications: PBAccessoryDidConnectNotification PBAccessoryDidDisconnectNotification The application can register for these notifications by adding itself as an observer on the NSNotificationCenter object: [[NSNotificationCenter defaultCenter] addObserver:myObject selector:(mySelector:) name:PBAccessoryDidConnectNotification object:[PBAccessory sharedClass]];
